The game I created for this project is a game I refer to as Savant which is a text based adventure game involving a weak gangly knight who has been tricked into saving the Princess Amber who has been abducted by a dragon named Emrys. This knight has been tricked into saving the princess alone and in the game there are four rooms including the start room.

Generally this game involves a few puzzles and is set up in a specific way so that to succeed in one room you must have visited another room prior. However, other puzzles simply involving some abstract thinking.
